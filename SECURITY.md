# Security Policy

## Reporting a Vulnerability
If you discover a security vulnerability, please report it responsibly by emailing the project maintainer. We will respond as quickly as possible.

## Supported Versions

| Version | Supported          |
|---------|-------------------|
| Latest  | :white_check_mark: |
| Older   | :x:               |

## Security Updates
We strive to promptly address security issues. Please check the [releases](./RELEASES.md) for updates.

## Responsible Disclosure
Please do not publicly disclose vulnerabilities until they have been addressed.
