# Contributing to Archivum 📚

Thank you for your interest in Archivum! We are building a federated library ecosystem that bridges the gap between physical books and digital experiences.

## 🏗 Project Architecture
Archivum is organized as a monorepo:
- **/apps**: Frontend applications
- **/services**: Backend services
- **/packages**: Shared logic and UI components.
- **/docker**: Deployment configurations for self-hosting.
